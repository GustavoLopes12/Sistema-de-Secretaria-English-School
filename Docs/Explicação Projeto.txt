Oque é:
O ManagerES é um sistema para secretaria de uma escola fictícia de inglês poder executar suas tarefas 
dentro do sistema escolar de forma mais fácil e rápida. O projeto na parte front-end foi desenvolvido em html, css 
e javascript, no back-end desenvolvi uma rest-api em NodeJs com framework ExpressJs utilizando o banco de dados MySql.
Esse projeto foi realizado com objetivo de aplicar oque aprendi durante os estudos sobre NodeJs, tinha o objetivo de 
realizar algo útil e desse objetivo nasceu este projeto, ele futuramente pode evoluir com mais funcionalidades para 
secretaria tanto quando a integração com outros sistemas para professores, alunos e diretoria.

Tecnologias Usadas:
	- Draw.io foi o site utilizado para fazer a modelagem UML do banco de dados da escola;
	- MySql foi o banco de dados escolhido para armazenamento dos dados;
	- NodeJs foi a ferramenta escolhida para desenvolvimento do backend do projeto;
	- RestAPI é o modelo de arquitetura utilizado no projeto visto que ele será utilizado na web;
	- ExpressJS é o framework do nodejs utilizado para o desenvolvimento da restapi;
	- Sucrase foi um framework utilizado para o padrao de imports e exports do projeto ser JavaScript e não CommonJS;
	- Nodemon foi o framework utilizado para carregar o projeto Node, sem necessitar de ficar subindo o server a cada mudança de codigo.
